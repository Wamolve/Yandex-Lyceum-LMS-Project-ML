## 🚀 Key Pipeline Steps

1. **Exploratory Data Analysis (EDA):** Analyzed feature distributions, correlation matrices, and identified missing values/outliers.
2. **Data Preprocessing:** Handled missing data, performed feature scaling, and encoded categorical variables (using One-Hot / Target encoding).
3. **Feature Engineering:** Created new domain-specific features to boost the predictive power of the models.
4. **Modeling & Tuning:** Tested multiple algorithms (such as Random Forest, Gradient Boosting), performed cross-validation, and optimized hyperparameters.
5. **Ensembling / Optimization:** Combined models and optimized decision thresholds to generate the final improved predictions.
